# Journex Website

A beautiful, modern website for the Journex journaling application.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Smooth Animations**: Elegant scroll animations and transitions
- **Modern UI**: Clean design matching the desktop app's aesthetic
- **Fast Loading**: Optimized HTML, CSS, and JavaScript

## Color Theme

The website uses the same warm, elegant color palette as the desktop app:
- Background: `#FAF7F2` (Warm cream)
- Text Primary: `#2C2416` (Dark brown)
- Accent: Soft lavender/pink for highlights
- Cards: White with subtle shadows

## Structure

- `index.html` - Main HTML structure
- `style.css` - All styling and animations
- `script.js` - Interactive features and scroll effects

## Sections

1. **Hero** - Large logo, tagline "Chronicle Your Odyssey", and CTA buttons
2. **Features** - 8 feature cards showcasing app capabilities
3. **Download** - Platform-specific download options
4. **About** - Information about Journex
5. **Contact** - Contact information and social links
6. **Footer** - Copyright and tagline

## Deployment

Simply upload the `website` folder to any web hosting service:
- GitHub Pages
- Netlify
- Vercel
- Traditional web hosting

## Customization

To update the logo, replace the image path in `index.html`:
```html
<img src="path/to/your/logo.png" alt="Journex Logo">
```

## Browser Support

- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers
